# 🚢 LLM-RuleVST

## 📌 Overview

This repository is for the paper:

**LLM-RuleVST: A Rule-Guided Visual-Sequence Transformer Framework for UAV-Based Vessel Trajectory and Risk Prediction in Inland Waterways**

The codebase is currently being organized and improved. At this stage, only part of the UAV visual perception module is included.

---

## 🚧 Current Status

This repository is still under preparation.

Currently available:

* YOLO-based UAV vessel detection module
* Basic training and inference scripts
* Preliminary support for UAV-based inland waterway vessel perception

Other modules are still being organized and will be updated later.

---

## 🚀 Usage

### Training

```bash
python train.py --config config.yaml
```

### Inference

```bash
python infer.py --weights best.pt --source test_images/
```
